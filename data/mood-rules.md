# 心情-音乐映射规则

## 天气映射
| 天气 | 推荐风格 | 示例 |
|------|---------|------|
| 晴天/温暖 | 轻快流行、独立民谣 | 轻快的吉他、阳光氛围 |
| 阴天 | 舒缓流行、R&B | 柔和的人声、温暖旋律 |
| 雨天 | 慵懒民谣、Lo-fi、慢歌 | 雨声氛围、安静钢琴 |
| 雪天 | 温暖治愈、经典情歌 | 温暖的声音、节日气氛 |
| 大风/极端 | 室内 cozy 氛围、放松纯音乐 | 安全的室内感 |

## 时间映射
| 时段 | 推荐风格 | 节奏 |
|------|---------|------|
| 06:00-09:00 早晨 | 轻快流行、民谣、古典 | 中速，唤醒 |
| 09:00-12:00 上午 | Lo-fi、纯音乐、轻爵士 | 舒缓，专注 |
| 12:00-14:00 午间 | 轻松流行、独立音乐 | 中速，放松 |
| 14:00-18:00 下午 | 有活力的流行、摇滚 | 中快，提神 |
| 18:00-21:00 傍晚 | R&B、爵士、民谣 | 中慢，放松 |
| 21:00-06:00 深夜 | 氛围音乐、轻爵士、慢歌 | 慢速，助眠 |

## 心情映射
| 用户表达的心情 | 推荐方向 |
|---------------|---------|
| 开心/兴奋 | 配合开心的能量，选有节奏感的歌 |
| 难过/emo | 先认同情绪，选有共鸣的抒情歌，再慢慢转到温暖的歌 |
| 焦虑/压力 | 选安静舒缓的纯音乐或 Lo-fi，帮助放松 |
| 无聊/想听歌 | 根据时间和天气主动推荐，带点惊喜感 |
| 怀旧 | 选 2000-2010 年代的华语经典 |
| 需要能量 | 选节奏感强的流行或摇滚 |
